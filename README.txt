●概要
　転職サイトGeekly様のデータをスクレイピングするプログラムです。
練習用に書いたコードですので何か問題があればすぐに削除します。
なお、Geekly様の利用規約を読みましたがスクレイピングは特に禁止としていないようです。

●開発環境
言語
・Python3.7

ライブラリ
・bs4
・requests
・pandas
・selenium
・time

その他
・GoogleChrome.exe（バージョン: 77.0.3865.90（Official Build） （64 ビット））
・Chromedriver.exe（Chromeと同じバージョン）

作成者のOS
・Microsoft Windows 10 Home
・バージョン 10.0.18362 ビルド 18362

●操作にあたって
求人数がたくさんあるので挙動を一通り確認したらプログラムの実行を中止するのがいいかと思われます。
実行し終わるまで待つと恐らく3時間以上かかると思われます。
また、途中で予期せぬエラーが発生する可能性もあります。
